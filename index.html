What is this?

This is a simple experiment which displays the current live subscriber count on YouTube for a channel or user. The count displayed on YouTube is often incorrect because it doesn't update in real time. The count on this page is taken directly from Google's Infrastructure and hence is as accurate as possible.

You can enter the new channel name in any format. You can also input search terms for a channel, this will also try to automatically correct spellings in most cases. This is as good as YouTube Search itself, because it uses the same engine to search.

You can literally enter anything that YouTube supports. Examples:

  • Channel Username (Example: PewDiePie)
  • Channel ID (Example: UCKlhpmbHGxBE6uw9B_uLeqQ)
  • Channel Name (Example: Fine Brothers Entertainment)
  • YouTube URL (Example: youtube.com/mkbhd)
  • Search Terms (Example: Linus)
  • Self Discovery Channels (Example: #Music)
